# Initialize Production-Level NestJS Project

Initialize a clean, production-ready NestJS project using the Nest CLI inside a specified folder, install essential dependencies, and scaffold a strong base architecture suitable for large-scale applications.

## Input

<project-name> = the name of the nestjs application and folder lowercase with dashes (e.g. nest-backend-service)

## Tasks to Perform

1. Check if @nestjs/cli is installed gloablly if not then install it using: npm i -g @nestjs/cli command
2. Verify installation again: nest --version
3. Create New NestJS Application nest new <project-name>
4. cd <project-name>
5.
